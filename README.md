# UOS
OS for ESP8266 based Smartwatches

## Current Features
- Stopwatch
- Internet Clock
- Timer
- Click Counter
- Flappy Bird
- Pong
- eReader
- Battery Monitor
- WiFi Scanner
- WiFi Monitor
- Packet Monitor
- Resource Monitor
- OTA Update

## System Options
- Restart
- Light Mode / Dark Mode
- WiFi Manager
- Dim Display / Night Mode
